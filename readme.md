## draw
![Image](img/draw.png)  
**"a simple drawing tool"**  
version: `0.9`  
needs: `python3, pygame`  
usage: `draw <file>`  

features:
* generates random filenames for lazy people.
* only works with square images.
* supports 8 different colors.
* highly customizable code.
